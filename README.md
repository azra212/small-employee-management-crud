# Employee Management App

A small full-stack CRUD application built as a personal learning project.

The goal of this project was to practice working with a modern JavaScript stack outside of my usual WordPress development workflow, with a focus on React, API communication, PostgreSQL, form handling, and clean error feedback.

## Overview

The app allows basic employee management:

- Add new employees
- View employee records
- Update employee details
- Delete employees
- Validate user input
- Display friendly success and error messages

## Tech Stack

**Frontend**

- React
- TanStack Query
- Chakra UI
- React Toastify

**Backend**

- Node.js
- Express
- PostgreSQL

## What I Practiced

- Building a REST API with Express
- Connecting a React frontend to a backend API
- Managing server state with TanStack Query
- Handling PostgreSQL validation and constraint errors
- Displaying cleaner user-facing error messages
- Structuring a small full-stack project

## Notes

This is intentionally a small project, created for learning and experimentation.

My main professional background is WordPress development, so this project was a way to explore a different stack and strengthen my general full-stack JavaScript skills.

## Running Locally

Install dependencies:

```bash
npm install
```

Start the backend and frontend according to their respective project folders.

Make sure PostgreSQL is running and your environment variables are configured before starting the server.
